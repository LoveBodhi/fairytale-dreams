# fairytale-dreams

“多彩童话梦”，是童话主题的开源TheXTech游戏组件的最小集。<br>
"Fairytale Dreams", is a minimal TheXTech game asset that features Fairy Tales.

本组件的部分会向后移植到“多彩童话仙境”。<br>
Part of this asset could be backport to "New Fairytale Wonderland".
